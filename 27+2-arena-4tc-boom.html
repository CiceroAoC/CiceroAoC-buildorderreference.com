27+2 Arena 4TC Boom

Only use when no possibility of needing to defend - this is an important build for pocket position in Arena teamgames. This build has a later Imperial time than a 3TC boom, but is needed for the more expensive unit compositions. This version has you get heavy plow before building the 4th TC, but some prefer to delay this until having ~30 farms.
