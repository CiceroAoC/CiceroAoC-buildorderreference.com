Interactive build order tutorials and practice scenarios for Age of Empires II DE. Available as two mods (well soon enough anyway):

Interactive Build Order Guide

Interactive Build Order Guide is a series of interactive tutorials for introducing players to some of the most important build orders for the game. The 12 scenarios included in this mod each feature step-by-step instructions in both text and audio, with voicework by ZeroEmpires. They also feature a scoring system that marks the player at several benchmarks, leading to a final grade (A+ to F). See the FAQ page for more details.

Build Order Challenges *COMING SOON*

Features ?? scenarios not available in my other mod. Build Order Challenge scenarios are a step up from my previous ones as they don't feature any instructions. Instead, go to the builds page to see my suggested builds for completing these challenges. As before, you will be graded on a scale of A+ to F - some of these scenarios are much more difficult however! These are intended to be a fun way of motivating your build order practice.
